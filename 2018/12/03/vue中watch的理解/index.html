vue中watch的理解

Posted on 2018-12-03

首先确认 watch 是一个对象，一定要当成对象来使用，对象就有键，有值；

键：就是你要监听的那个值，比如说$route,这个就是要监听路由的变化。或者说是data中的某个变量；

值：

1.值可以是函数：当你监听的值发生变化时，需要执行的函数，这个函数有两个形参，第一个是当前值，第二个值是变化后的值。

2.值也可以是函数名：不过这个函数名要用单引号来包裹；

3.值可以是包括选项的对象：（handler，deep，immediate）

  1. 第一个handler：其值是一个回调函数，即监听到变化时应该执行的函数；
  2. 第二个是deep：其值是true或false；确认是否深度监听。（一般监听时是不能监听到对象属性值的变化的，数组的值变化可以听到）

  3. 第三个是 immediate：其值是true或false；确认是否以当前的初始值执行handler的函数；

    export default {
name: "homePage",
data(){
return{
 a:1,
 b:2,
 c:3
}
},
watch:{
a:function (val,oldVal) {
  console.log("a当前值"+ val ,"a变化后的值"+ oldVal );
},
//方法名
b:'changeB',

//选项的对象
c: {
 handler: function (val, oldVal) {
    console.log("c变化了")
 },
 deep: true,
 immediate: true
}
},
created(){
this.a = this.a + 1;
this.b = this.b + 2;
},
methods:{
changeB:function (val,oldVal) {
console.log("当前值"+ val ,"变化后的值"+ oldVal );
}
}
}
