websrorm下配置less环境 and 设置压缩css

首先,webstorm这个强大的编辑器是自带less的，不过编译过程是需要nodejs环境。

步骤

  1. 下载对应版本的NODEJS： HTTPS://NODEJS.ORG/。

    1. 安装完成后打开命令符(WIN+R）,分别输入NODE-V以及NPM-V，如果返回版本号说明你已经安装成功了。如下图：
    2. 安装Less：npm install less。
    3. 配置webstorm：file ->settings ->Tools ->File Watchers,点击右上角的“加号”标志add，进入对话框，如下图：
    5. program里设置的是：lessc编译路径，它会默认编译到根目录。例如：/Users/xx/node_modules/less/bin/lessc。
    6. 如果你想顺便压缩编译后的css，除了在Arguments中写入–plugin=less-plugin-clean-css外还要安装压缩插件：
      npm install -g less-plugin-clean-css，不过这里需要注意的是：less-plugin-clean-css的路径和lessc编译路径的/Users/xx/node_modules/less应一致；
    7. 你可以为你的编译器指定一个快捷键，然后你可以为这个编译器指定一个快捷键，我使用的是ALT + B；也可以手动右键css ->less；如下图：
    9. 之后打开一个.less后缀名的文件，快捷键编译，如果webstorm底部状态控制器显示 Process finished with exit code 0 ，那么恭喜你，大功告成！

    小问题

  1. 明明已经安装nodejs了，但总提醒你nodejs不存在，这个时候可以看一下你的nodejs安装的路径。

  1. 出现乱码提示的问题，说明nodejs没有安装成功或者安装有问题，解决办法忒粗鲁，卸载重新安装nodejs，安装好后切记重启电脑；
